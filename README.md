# 小屿和 - 智能疗愈伴侣 (Xiaoyu And)

**版本**: v2.0.0 (Incense First)

> "推门见山，一触即愈"

本项目为「小屿和香」品牌旗下数字化伴侣应用。区别于传统 IoT 设备，我们强调**物理世界的优先性**。用户的使用旅程始于"点燃线香"这一物理仪式，App 仅作为增强体验的"伴侣"（Companion），通过 NFC 一触即连，提供沉浸式的视听疗愈体验。

## 核心理念 (Core Philosophy)

*   **Incense First (线香即入口)**: 物理点香动作触发数字体验，而非 App 遥控物理设备。
*   **One-Tap Connection (一触即连)**: NFC 极速唤醒，解决蓝牙配对痛点。
*   **Layered Immersion (分层沉浸)**: 叙事音频 (Layer 1) + 可调白噪 (Layer 2) 的双层听觉设计。
*   **Zen Bottle (回响漂流瓶)**: "写一换一"的极简社交，提供安全的倾诉空间。

## 核心功能 (Features)

1.  **NFC 唤醒 & 极简选香**: 
    -   手机触碰香插，自动进入香型确认页。
    -   **查看香方**: 点击 "i" 图标或 "查看香方" 提示，探索制香师背后的故事与配方细节。
    -   **即刻点燃**: 点击 **"燃起 · 此刻" (Ignite · This Moment)**，伴随 🔥 视觉动效，直接进入沉浸疗愈。
2.  **沉浸式疗愈播放器 (Immersion)**:
    *   **Layer 1**: 绑定香型的叙事背景音（如古寺钟声、深海雨声）。
    *   **Layer 2**: 用户可调节的功能底噪（白噪音/粉噪/红噪），满足专注、助眠、放松需求。
    *   **Visual**: 配合呼吸节奏的视觉微光与动态烟雾引擎。
3.  **树洞与漂流瓶**: 疗愈结束后的情绪记录与匿名共鸣。

## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
